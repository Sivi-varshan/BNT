Project

Pathfinding Visualizer Using React

This is a web-based iteractive pathfinding visualizer developed using React, TypeScript, and Tailwind CSS. The tool allows users to create custom mazes and simulate various pathfinding algorithms including Dijkstra’s, A*, Breadth-First Search (BFS), and Depth-First Search (DFS). It offers a user-friendly interface that aids in visual learning and comparison of different algorithmic strategies. The project demonstrates both frontend development skills and a strong understanding of algorithmic logic and visualization techniques.
